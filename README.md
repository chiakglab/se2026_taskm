# タスク管理アプリ（taskm）

音楽フェスのタスク管理アプリです。
誰が何を担当しているか、進捗はどうかを一覧で確認できます。

## 機能

- スタッフの登録・管理
- タスクの作成・編集・削除
- 担当者・締切を指定してタスクを割り当て
- ステータス管理（未着手・進行中・完了）
- 締切が近い・過ぎているタスクの自動ハイライト

## 動かし方

```
pip install -r requirements.txt
python app.py
```

ブラウザで `localhost:5555` を開く。

最初はスタッフが0人なので、まず「スタッフ管理」からスタッフを追加してから、
タスクを作成すること。
